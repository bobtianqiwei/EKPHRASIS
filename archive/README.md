# Archive

This folder holds old or unused assets that are not required to run EKPHRASIS.

- **demo_backup/** – Earlier interface demos and prototypes (e.g. 1021demo, 1125demo, block_generator, centered-functional-interface).
- **Composition Classification Model Bob Oct 20 2024.ipynb** – Exploratory training/prediction notebook (different data path; model is now trained via `ml/train_and_save_model.py`).
- **ml_test_dataset/** – Test data used for experiments (e.g. Flower).
- **composition_model.h5**, **visual_balance_model.h5** – Previous model checkpoints; current models live in `ml/models/`.

You can delete or move anything here without affecting the main app.
